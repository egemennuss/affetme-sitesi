# Bana Affetsen? 💕

Sevgiyle yapılmış, animasyonlarla dolu romantik bir affetme sitesi!

## 🎨 Özellikler

- ✨ **Harika Animasyonlar** - Efektli parçacık patlaması, sparkle, kalp yağmuru
- 💕 **Romantic Design** - Modern ve tatlı arayüz
- 🔐 **Kilit Sistemi** - Telefon numarası doğrulaması ile gizli mesaj
- 📱 **Mobile Optimized** - iPhone 15 Safari ve mobil Google için optimize
- 🎯 **Touch Friendly** - Tüm cihazlarda sorunsuz çalışır
- 🚀 **GitHub Pages Ready** - Direkt deploy edilebilir

## 🚀 Hızlı Başlangıç

1. Dosyaları indirin
2. `index.html` dosyasını tarayıcıda açın
3. Bütün efektleri ve animasyonları keyfini çıkarın!

## ⚙️ Özelleştirme

`index.html` dosyasında şu değişiklikleri yapabilirsiniz:

- **Telefon Numarası**: Dosyada `checkPhone()` fonksiyonunu bulup telefon numaranızı girin
- **Mesajlar**: `noMessages` arrayinde özel mesajlar ekleyin
- **Renkler**: CSS'deki gradient renklerini değiştirin
- **Metinler**: İstediğiniz herhangi bir yeri özelleştirin

## 📱 Tarayıcı Desteği

- ✅ Safari (iOS 14+)
- ✅ Chrome (Android 8+)
- ✅ Firefox
- ✅ Edge
- ✅ Google Lens (Mobil)

## 🎯 GitHub Pages'e Deploy Etme

```bash
# 1. GitHub'da yeni repo oluşturun
# 2. Repository'i klonlayın
git clone https://github.com/YOUR_USERNAME/affetme-sitesi.git
cd affetme-sitesi

# 3. Dosyaları ekleyin
git add .
git commit -m "İlk commit - Affetme sitesi"

# 4. GitHub'a push edin
git push -u origin main

# 5. GitHub'da Settings → Pages → Branch: main seçin
# 6. Bitiş! Siteniz https://YOUR_USERNAME.github.io/affetme-sitesi adresinde live
```

## 💡 İpuçları

- Sitede "Affetim" butonuna basarak müthiş efektler görebilirsiniz
- "Biraz daha" butonuna her tıklamanızda kaçan buton daha zor olacak
- Telefon numarası doğru girince gizli mesaj sayfası açılır
- Sayfayı telefonunuzda açıp arkadaşınıza gösterebilirsiniz!
